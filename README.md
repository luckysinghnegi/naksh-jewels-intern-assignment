# Naksh Jewel – Full Stack Project

Naksh Jewel is a full-stack web application built using the MERN stack.  
The project includes a modern React frontend, a Node.js + Express backend, and Docker support for easy setup and deployment.

This project was created as part of practical learning and real-world full-stack development practice.

---

## Tech Stack

**Frontend**
- React (Vite)

**Backend**
- Node.js
- Express.js

**DevOps**
- Docker
- Docker Compose

---

## Project Structure

Naksh_Jewel/

├── frontend/ # React frontend

---
## Environment Variables

### Backend (.env)
PORT=3000

### Frontend (.env)
VITE_API_BASE_URL=http://localhost:3000

---
## Running with Docker

Make sure Docker Desktop is running.

From the **root folder**:docker compose up --build F

Frontend will run at: http://localhost:5173
Backend will run at: http://localhost:3000
